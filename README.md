# scarletzhang.github.io
Thoughts, sketches, and lessons from my journey in front-end, product design, and building things people love.
[View Blog](https://scarletzhang-z.github.io/blog/)

## installation
1. Make sure you have [Jekyll](https://jekyllrb.com/docs/installation/) installed.
2. Clone the repository: `git clone https://github.com/scarletzhang-z/scarletzhang-z.github.io.git`
3. Navigate to the directory: `cd scarletzhang-z.github.io`
4. Install dependencies: `bundle install`
5. Run the site locally: `bundle exec jekyll serve`
